# 人脸五官检测系统 🧬

使用 **MediaPipe + Vue3** 实现的免费开源人脸检测工具

## ✨ 功能

- 📸 上传人物照片
- 🔍 自动检测五官位置（眼睛、鼻子、嘴巴、脸部轮廓）
- 📊 在图片上标注虚线和关键点
- 📈 显示检测数据（坐标、尺寸等）

## 🚀 快速开始

### 1️⃣ 安装依赖
```bash
npm install
```

### 2️⃣ 启动开发服务器
```bash
npm run dev
```

浏览器会自动打开 `http://localhost:5173`

### 3️⃣ 使用流程
1. 点击"选择照片"按钮上传包含人脸的图片
2. 点击"开始检测"按钮
3. 等待模型分析（首次使用需要加载模型，约 5-10 秒）
4. 查看结果图片和检测数据

## 📦 项目结构

```
.
├── index.html          # HTML 入口
├── package.json        # 项目配置
├── vite.config.js      # Vite 配置
└── src/
    ├── main.js         # Vue 应用入口
    ├── App.vue         # 主应用组件
    └── utils/
        └── faceDetector.js  # MediaPipe 检测逻辑
```

## 🔧 技术栈

- **Vue 3** - 前端框架
- **Vite** - 构建工具
- **MediaPipe** - 人脸识别模型（免费开源）
- **Canvas** - 绘制标注

## 📝 检测数据说明

- **左眼/右眼** - 眼睛外角坐标
- **鼻尖** - 鼻子最下点坐标
- **嘴巴** - 嘴巴中心坐标
- **脸宽/脸高** - 脸部总体尺寸（像素）

## ⚠️ 注意事项

- 首次使用需要下载模型文件（约 50MB），请保持网络连接
- 最好在桌面浏览器上使用
- 图片中人脸要清晰可见，建议脸部占图片 30% 以上
